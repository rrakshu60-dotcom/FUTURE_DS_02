Task 2: Customer Retention & Churn Analysis

Tools Used:
Power BI

Dataset:
Telco Customer Churn Dataset

Description:
Analyzed customer churn patterns to understand why customers leave and identify retention opportunities.

Key Insights:
• Month-to-month contract customers show the highest churn.
• Customers using electronic check payment method churn more.
• Customers with lower tenure tend to leave earlier.
• Higher monthly charges slightly increase churn probability.

Deliverable:
Interactive Power BI dashboard showing churn patterns and customer retention insights.
